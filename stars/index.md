## Ninja Stars

ReviewNinja understands that often in code review what matters most is *who* did the review. By starring a Pull Request, it shows that the reviewer has seen the code change and has approved it.

To star a pull request, click the ninja <i class="fa fa-star ng-scope"></i>.

{% image star-people.png width="30%" %}

### Ninja Star Threshold

You can define the number of ninja stars required for your project in order for the Pull Request to be deemed "successful".

To configure this setting, click <i class="fa fa-cog"></i> on the repo. You must specify a number between 0 and 99.
